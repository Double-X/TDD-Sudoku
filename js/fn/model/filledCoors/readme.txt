FMBaseFilledCoors is responsible for returning the list of coordinates of the initially filled coordinates varied by the character value
spacing only.
FMFilledCoors is responsible for returning the list of coordinates of the initially filled coordinates with only the row groups, column
groups, rows in the same row group and columns in the same column group swapped by the swapping combinations.
FMNewFilledCoors is responsible for returning the list of coordinates of the initially filled coordinates with all the above variations.
