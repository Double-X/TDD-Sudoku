FMAPIs is responsible for exposing all Model APIs to Controller.
FMFilledCoorNoMinMax is responsible for the valid range of the number of initially filled coordinates.
FMInputViolations is responsible for returning all messages describing why the input's invalid(having none of them means it's valid).
FMIsWon is responsible for checking whether the board's solved.
FMJSONIO is responsible for accessing the IO of the recorded demo file contents.
FMNewData is responsible for using the deterministic data to return nondeterministic counterparts.
FMPickedCombination is responsible for returning a speficied combination of elements within a list.
FMRecDemo is responsible for recording the demo file contents.
