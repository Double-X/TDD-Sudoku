FVPlayDemo is responsible for playing the loaded demo file contents.
FVRenderBoard is responsible for exposing all the View APIs.
FVRenderDoms is a seam, which facilitates unit testing, for actually rendering doms.
FVRenderedDoms is a seam, which facilitates unit testing, for actually getting the current dom states.
FVShowMsg is responsible for showing all modal messages.
FVVisibility is responsible for controlling the visibility of all views.
