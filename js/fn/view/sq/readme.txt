FVRenderedFixedSq is responsible for returning the rendered read only square that is initially filled with characters.
FVRenderedSq is responsible for returning the rendered square that accepts user inputs.
FVRenderedSqs is responsible for returning the specified portions of the 2D board.
FVRenderSqParams is responsible for encapsulating low level configurations as if they were constants.
FVRenderSqs is responsible for drawing the new 2D board composed by squares.
